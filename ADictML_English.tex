\begin{abstract}
This dictionary of machine learning evolved during the design and implementation of the courses 
CS-E3210 {\bf Machine Learning: Basic Principles}, CS-C3240 {\bf Machine Learning}, CS-E4800 {\bf Artificial Intelligence}, CS-EJ3211 {\bf Machine Learning with Python}, CS-EJ3311 {\bf Deep Learning with Python}, CS-E4740 {\bf Federated Learning}
and CS-E407507 {\bf Human-Centered Machine Learning} offered to Finnish university 
students via {\bf Aalto University} \url{https://www.aalto.fi/en}, to Finnish adult learners 
via {\bf The Finnish Institute of Technology (FITech)} \url{fitech.io} and to international students 
via the {\bf European University Alliance Unite!} \url{https://www.aalto.fi/en/unite}. 
\end{abstract}
